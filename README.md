## mini_project_03 - Angelo Morelli
- A simple todo list built with REACT.
## Live Link:
- https://exquisite-melba-36c274.netlify.app/
## Components:
# App.jsx
- The main application component which manages task state (tasks, title, description), handles list functionality (adding, deleting, modifying, and completing tasks), handles local storage (save and load tasks) and renders the input form and task list.
# ListItem.jsx
- The component which displays list items with a checkbox, the task title and description, edit/delete buttons, and save/cancel buttons while editing.
